# D3 Linechart

### Data format

Data is expected to be in `"Long"` format, i.e. an array of JSON objects:

```json
[
    {
    "Date": "2016-01-01",
    "count": 1,
    "group": "A"
    },
    {
    "Date": "2016-01-01",
    "count": 3,
    "group": "B"
    },
    {
    "Date": "2016-01-02",
    "count": 2,
    "group": "A"
    },
    {
    "Date": "2016-01-02",
    "count": 12,
    "group": "B"
    },
    ...
]
```
